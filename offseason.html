Installation

To get Off Season up and running on your computer:

  1. Download the game file above. It will download in zip format. You can play the game without unzipping the game file!
  2. Install a GameBoy Advance emulator of your choice. The emulator is the program that will allow you to open and run Off Season. There are a variety of emulators out there. I would recommend VisualBoy Advance if you're on a PC and OpenEmu if you're on a Mac. You can download either emulator above!
    • To install VisualBoy Advance, unzip the download file and BOOM! You're done. To run VisualBoy, simply double-click the executable inside the unzipped folder!
    • To install OpenEmu, unzip its download file. Drag OpenEmu.app into your applications folder in Finder, and BOOM! You're done.
  3. Run Off Season from the emulator.
    • On VisualBoy Advance, click "File" "Open" in the navigation menu and select OffSeason.zip.
    • On OpenEmu, click "File" "Add to Library..." and select OffSeason.zip.
